## 使用說明

使用者僅完善 Config.json 即可

__Other_Setting 是 角色額外設定

__Reference 關起來則會關掉回應回覆功能

__OtherUser_Call 就是非主人的其他暱稱，例如 Haroiii先生，Haroiii 

最後，windows 請將 Start_bot.sh 改為 Start_bot.bat

Liunx/Unix 環境使用者

```bash
chmod 777 ./Start_bot.sh
```

再來

```bash
./Start_bot.sh
```
